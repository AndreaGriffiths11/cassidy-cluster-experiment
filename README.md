# Cassidy Cluster Experiment

JavaScript implementation of Cassidy Williams' `cassidyCluster` tile clustering algorithm.

## Decision flow

```mermaid
flowchart TD
    A[Start with a copy of the input list] --> B[Look at the right edge]
    B --> C[Find the rightmost cluster]
    C --> D{Does the same value exist to its left?}
    D -- Yes --> E{Is this cluster already before completed clusters?}
    E -- No --> F[Reverse from the matching value through the end]
    E -- Yes --> G[Reverse from this cluster through the end]
    F --> H[Record the new state]
    G --> H
    H --> B
    D -- No --> I[Move left to the next cluster]
    I --> J{Any incomplete cluster left?}
    J -- Yes --> C
    J -- No --> K[Return the clustered list]
```
